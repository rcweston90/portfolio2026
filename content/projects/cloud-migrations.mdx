## Overview

As Senior Product Designer for Cloud Migrations at Atlassian, I co-led a team responsible for driving Server End-of-Support awareness and Cloud Migration activation, developing quarterly roadmaps and introducing practices that improved project outcomes.

## The Challenge

Helping enterprise customers understand the need to migrate from Server to Cloud, while making the migration process as smooth as possible.

## The Solution

By developing clear communication strategies, user-centered design approaches, and data-driven roadmaps, we were able to effectively drive awareness and activation.

## Impact

- Co-led team driving Server End-of-Support awareness and Cloud Migration activation
- Developed quarterly roadmaps
- Introduced practices that reduced project churn by 20%

## Key Learnings

Enterprise migrations require careful consideration of user needs, business requirements, and technical constraints. Clear communication and strategic planning are essential for success.
